## 📁 Projetos

### 🖩 Calculadora Web
- **Tecnologias:** HTML, CSS, JavaScript
- **Descrição:** Calculadora interativa que realiza operações de soma, subtração, multiplicação e divisão diretamente no navegador.
- **Link:** [Acessar projeto](https://ian-pedro-passos.github.io/portfolioHUB/calculadora/calculadora.html)

### 👤 Sistema de Cadastro
- **Tecnologias:** Java
- **Descrição:** Programa de console que cadastra pessoas, identifica a maior e menor altura e conta quantos homens e mulheres foram registrados.
